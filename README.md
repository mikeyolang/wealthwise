# 🏦 Wealthwise: AI-Powered Personal Finance

Wealthwise is a premium, **production-ready Flutter application** that redefines personal finance management. Powered by **Google Gemini 1.5 Flash AI**, it provides smart insights, automated tracking, and goal-oriented saving features with a stunning dark-first fintech aesthetic.

---

## 🎨 Design & Aesthetic

Wealthwise is built for users who value a **clean, premium, and modern** interface:
- **Dark Mode First**: Deep navy/charcoal tones for a sleek fintech experience.
- **Vibrant Accents**: Emerald for income, Coral for expenses, and Gold for goals.
- **Smooth Animations**: Powered by `flutter_animate` for a truly dynamic feel.
- **Glassmorphism**: Subtle translucent effects on dashboard cards for visual depth.

---

## 🚀 Key Features

### 🏦 Smart Dashboard
- **Net Balance Visualization**: Large, animated balance cards with income/expense summaries.
- **AI Daily Tip**: Fresh financial advice generated by Gemini AI on every app launch.
- **Spending Breakdown**: Visual charts representing category-wise expenditure.

### 📝 Seamless Transactions
- **Offline-First**: All data is stored locally in SQLite and synced to the cloud in the background.
- **Quick Logging**: Add transactions with merchant names, categories, and payment methods.
- **Search & Filter**: Find any transaction with advanced search and filtering.

### 🎯 Financial Goals
- **Savings Targets**: Set and track progress towards personal financial milestones.
- **Deep Progress Tracking**: Goal detail views with circular progress indicators and contribution timelines.
- **Contribution History**: Keep track of every cent saved towards your dreams.

### 📊 AI Financial Reports
- **Gemini AI Insights**: Automated analysis of your spending habits and financial health.
- **Interactive Charts**: Line and bar charts showcasing trends and income vs. expense balance.

### 📄 Data Portability
- **CSV/PDF Export**: Professional financial reports generated on-demand and shareable via system share sheet.

### 🔒 Security & Sync
- **Biometric Authentication**: Fingerprint/FaceID lock for added privacy.
- **Cloud Synchronization**: Automatic, periodic background sync with Firebase Firestore.
- **Local Notifications**: Real-time alerts for transaction logs and goal milestones.

## ⚙️ Getting Started

### Prerequisites
1.  **Flutter SDK** installed on your system.
2.  **Firebase Project** set up (Google-services.json / GoogleService-Info.plist).
3.  **Gemini API Key** from [Google AI Studio](https://aistudio.google.com/).

### Installation
1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/wealthwise.git
    ```
2.  Install dependencies:
    ```bash
    flutter pub get
    ```
3.  Add your Gemini API Key in `lib/features/analytics/providers/analytics_provider.dart`.
4.  Run the application:
    ```bash
    flutter run
    ```
